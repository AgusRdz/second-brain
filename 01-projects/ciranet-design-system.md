# Ciranet Design System

## Overview

A Claude Code plugin within `dx-claude-code` that provides AI-assisted tooling for the Ciranet design system. Helps enforce component patterns, generate consistent UI code, and maintain design system documentation.

## Links

- **Repository**: dx-claude-code (GitLab)
- **Design system**: Part of Ciranet monorepo shared packages

## Purpose

- Enforce consistent component usage across the Ciranet monorepo
- Provide Claude Code with context about available design system components
- Generate UI code that follows established patterns
- Bridge between the design system library and AI-assisted development

## Integration Points

- Claude Code plugin system (CLAUDE.md conventions)
- Ciranet monorepo shared UI packages
- Angular Material theming
