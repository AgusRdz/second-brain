# brain

Capture engineering knowledge to the Second Brain Obsidian vault.

## Usage

- `/brain debug` — Create a debugging note from current session context
- `/brain decision` — Create a decision record
- `/brain til` — Create a TIL (Today I Learned)
- `/brain daily` — Create/update today's daily log
- `/brain project <name>` — Update a project note
- `/brain search <query>` — Search the vault for existing notes
- `/brain` — Auto-detect what's worth capturing from the session

## Triggers

This skill activates when you mention:
- /brain, capture to vault, save to brain, remember this
